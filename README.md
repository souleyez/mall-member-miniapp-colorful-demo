# 彩色手绘商场会员小程序演示版

这是从暗金版本机械复制后独立改造的纯前台微信小程序演示工程。新版本使用白底、薄荷绿与亮黄/粉/蓝/紫的手绘零售视觉，功能与原演示版保持一致，不会改动原工程。

## 工程信息

- AppID：`wx76e51a2cd77f5bcd`
- 工程名：`mall-member-colorful-handdrawn-demo`
- 演示入口：`pages/demo/demo`
- 兼容入口：`pages/home/home`、`pages/account/account` 等会重定向到对应演示页面

## 设计与边界

- 主风格稿由 GPT-image-2 参考客户截图生成，再转译成可运行 WXML/WXSS。
- 不把整张效果图放进包体，界面轮廓、色块与装饰主要由 CSS 构成。
- 不使用原小程序接口、域名、账户、支付或业务数据。
- 登录、积分、卡券、小票识别、停车缴费、报名、商户管理与兑换均为本地 Mock 交互。
- 页面不展示限制品牌名称或相关标识。

## 功能范围

单页容器包含 45 个演示视图，覆盖首页、找店、活动与内容、会员卡、成长值、积分、卡券、团购、在线商城、商品详情、礼遇商城、购物车、结算、订单、停车、导览、客服、礼宾借用、会员资料、车辆、设置、消息、商户管理与多商场聚合等流程。

所有可见操作入口都会产生页面切换、弹层、状态变化、Loading、成功提示或说明反馈。点击反馈仅做轻微明暗变化，不使用明显下压或缩放。

三个商场拥有独立的首页活动、会员体系、团购套餐与在线商城商品数据；全部内容均为本地演示数据。
